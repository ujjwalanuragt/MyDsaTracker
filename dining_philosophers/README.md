# Dining Philosophers Problem — PowerPoint Generator

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
cd dining_philosophers
python generate_presentation.py
```

## Output

`Dining_Philosophers_Problem.pptx` — A 13-slide academic presentation on the Dining Philosophers Problem in Operating Systems.

## Slides
1. Title Slide
2. Introduction
3. Problem Overview
4. Problem Scenario (with circular table diagram)
5. Resource Sharing Representation
6. Synchronization Challenges
7. Key Issues (Deadlock, Starvation, Mutual Exclusion)
8. Semaphore-Based Solution
9. Semaphore Operations
10. Pseudocode Using Semaphores
11. Problem with Basic Semaphore Solution
12. Deadlock Prevention Methods
13. Conclusion
